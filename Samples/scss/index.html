SCSS Hello World

安装

  1. 安装gulp和gulp-sass

    npm install gulp gulp-sass --save

  2. 添加gulp配置任务

    项目根目录新增gulpfile.js，内容如下：

    'use strict'; var gulp = require('gulp'); var sass = require('gulp-sass'); gulp.task('sass', function () { gulp.src('./css/*.scss') .pipe(sass().on('error', sass.logError)) .pipe(gulp.dest('./css/')); }); gulp.task('sass:watch', function () { gulp.watch('./css/*.scss', ['sass']) .on('change', function (event) { console.log('File ' + event.path + ' was ' + event.type + ', running tasks...'); });; });

  3. 创建VS Code任务

    在VS Code中，Ctrl+Shift+B添加Gulp, 在生成的.vscode/tasks.json中:

    { "version": "0.1.0", "command": "gulp", "isShellCommand": true, "args": ["--no-color"], "showOutput": "always", "tasks": [ { "taskName": "sass:watch", "isBuildCommand": true } ] }

  4. 运行任务

    Ctrl+Shift+B
